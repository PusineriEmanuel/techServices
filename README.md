## 🚀 Instalación y Uso

### Clonar el Repositorio

```bash
git clone [URL_DEL_REPOSITORIO]
cd techServices
```

### Ejecutar Localmente

1. Abrir `index.html` en tu navegador preferido
2. O usar un servidor local:

```bash
# Con Python
python -m http.server 8000

# Con Node.js (si tienes live-server instalado)
npx live-server
```
